# Text to Speech Converter

## Project Description

Text to Speech Converter is a web-based application developed using HTML, CSS, and JavaScript that converts written text into spoken audio using the Web Speech API. The application allows users to enter text, select their preferred voice or language, and customize speech settings such as rate and pitch to enhance the listening experience.

## ✨ Features

* Convert text into speech instantly
* Support for multiple languages and voice options
* Dynamic voice selection based on browser support
* Adjustable speech rate control
* Adjustable speech pitch control
* Pause and resume speech playback
* Stop speech playback at any time
* Simple, responsive, and user-friendly interface
* Real-time speech status updates

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Web Speech API

## ⚙️ How It Works

1. Enter text into the input area.
2. Select a preferred language or voice from the dropdown list.
3. Adjust the speech rate and pitch using the sliders.
4. Click the **Speak** button to convert text into speech.
5. Use the **Pause/Resume** button to control playback.
6. Click the **Stop** button to end speech output.

## Speech Controls

### Rate Control

Adjusts the speed of speech output.

* 0.5 → Slow
* 1.0 → Normal (Default)
* 1.5 → Fast
* 2.0 → Very Fast

### Pitch Control

Adjusts the tone of the generated voice.

* 0.5 → Low Pitch
* 1.0 → Normal (Default)
* 1.5 → High Pitch
* 2.0 → Very High Pitch
